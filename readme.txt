== Description ==
Adds placeholder text to Gravity Forms elements. The placeholder text is derived from the class name that you assign to a Gravity Forms element in the form settings page. For example, if you give an input element a class name of \"Enter your address\", then \"Enter your address\" will automatically become that elements placeholder text.

== Installation ==
Add the plugin to your WordPress installation. That\'s it. It will automagically work with Gravity Forms.
